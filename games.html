Fruit Field


Fruit Field is a token-collecting maze puzzle game. The objective of the game is to help Charlie the Cat collect all the fruits in the field and advance to next levels of the game, all while avoiding Dapper the Dog. The levels increase in difficulty as the game progresses.

This game was made using Java and Processing.
