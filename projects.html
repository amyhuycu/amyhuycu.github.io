PROJECTS


Fruit Field


Fruit Field is a token-collecting maze puzzle game. The objective of the game is to help Charlie the Cat collect all the fruits in the field and advance to the next levels of the game, all while avoiding Dapper the Dog.

Currently features 4 levels that increase in difficulty as the game progresses.

This game was made using Java and Processing.

Paint Bucket


Paint Bucket is a single-player sliding puzzle game. The objective is to catch all of the paint drops in the paint bucket with the respective color. The player uses the left and right arrow keys to maneuver the positions of the buckets.

Levels increase in difficulty as the game progresses. Features 4 different color themes.

This game was made using HTML and Javascript.
